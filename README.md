## 🚀 Featured Projects

### 🔥 Distributed Rate Limiter & API Gateway

A production-style API gateway built with **FastAPI, Redis, Docker, Kubernetes, Prometheus, and Grafana**.
Designed with token bucket and sliding window algorithms, Redis Lua scripts, metrics, tests, and benchmarking.

### 📈 Real-Time Financial Data Pipeline

An end-to-end data engineering project using **Kafka, Spark Structured Streaming, PostgreSQL, dbt, Airflow, Grafana, Power BI, and Docker**.
Built to simulate real-time market analytics and modern data platform workflows.

### 🧠 Mem0-Style Memory Layer for AI Agents

A long-term memory service for AI agents using **FastAPI, PostgreSQL, Redis, FAISS, LangChain, Docker, and Next.js**.
Built to store, retrieve, and manage contextual memory for intelligent applications.
