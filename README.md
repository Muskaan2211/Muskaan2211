## 🚀 Featured Projects

### 🔥 Distributed Rate Limiter & API Gateway

A production-style API gateway built with **FastAPI, Redis, Docker, Kubernetes, Prometheus, and Grafana**.

Designed with:

- Token bucket and sliding window rate limiting  
- Redis Lua scripts for atomic operations  
- Prometheus metrics and Grafana dashboards  
- Dockerized setup  
- Kubernetes manifests  
- Pytest test coverage  
- Benchmarking support  

---

### 📈 Real-Time Financial Data Pipeline

An end-to-end data engineering project built with **Kafka, Spark Structured Streaming, PostgreSQL, dbt, Airflow, Grafana, Power BI, and Docker**.

Designed to simulate a real-time financial analytics platform with:

- Streaming data ingestion  
- Real-time processing  
- Data transformation with dbt  
- Workflow orchestration with Airflow  
- PostgreSQL storage  
- Grafana monitoring  
- Power BI reporting  

---

### 🧠 Mem0-Style Memory Layer for AI Agents

A long-term memory service for AI agents built with **FastAPI, PostgreSQL, Redis, FAISS, LangChain, Docker, and Next.js**.

Designed to support:

- Memory storage  
- Semantic search  
- Vector-based retrieval  
- AI agent context management  
- Backend APIs  
- Web dashboard
